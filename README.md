# 💊 MedControl CLI – Controle de Medicamentos para Idosos

## 📌 Sobre o Projeto

O **MedControl CLI** é uma aplicação simples em linha de comando (CLI) desenvolvida com o objetivo de ajudar idosos, cuidadores e familiares a controlarem o uso correto de medicamentos no dia a dia.

O projeto foi desenvolvido como parte de um desafio prático individual com foco em boas práticas de desenvolvimento de software, incluindo versionamento, testes automatizados, documentação e integração contínua.

---

## 🎯 Problema Real

Muitos idosos precisam tomar vários medicamentos por dia, em horários diferentes. No entanto, é comum ocorrerem situações como:

- esquecimento de tomar o medicamento no horário correto;
- confusão entre medicamentos;
- dificuldade de organização por parte de idosos que moram sozinhos;
- dificuldade de acompanhamento por parte de familiares e cuidadores.

Esses problemas podem comprometer o tratamento médico, causar riscos à saúde e reduzir a qualidade de vida do paciente.

---

## 💡 Solução Proposta

A aplicação **MedControl CLI** permite que o usuário registre e acompanhe seus medicamentos de forma simples e rápida, diretamente pelo terminal.

Com essa ferramenta é possível:

- cadastrar medicamentos com horário;
- visualizar todos os medicamentos cadastrados;
- marcar medicamentos como tomados;
- armazenar os dados em arquivo para uso contínuo.

Mesmo sendo uma aplicação simples, ela resolve uma dor real e pode ser facilmente utilizada por estudantes, cuidadores ou familiares que queiram ajudar no controle de medicamentos de idosos.

---

## 👥 Público-Alvo

Este projeto foi pensado principalmente para:

- idosos que precisam organizar medicamentos;
- cuidadores de idosos;
- familiares que ajudam no controle de medicação;
- estudantes que desejam uma ferramenta simples de apoio à rotina de saúde.

---

## ⚙️ Funcionalidades

✔️ Cadastrar medicamento  
✔️ Definir horário de uso  
✔️ Listar medicamentos cadastrados  
✔️ Marcar medicamento como tomado  
✔️ Armazenamento em arquivo JSON  
✔️ Interface simples em linha de comando (CLI)
